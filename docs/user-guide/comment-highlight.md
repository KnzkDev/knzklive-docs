# コメントハイライト

コメントハイライトとは、配信者を支援する事で、一定期間コメント欄上部にバッジと名前のハイライトがつく機能です。

これは、配信者が有効化することで使用できる機能です。コメントハイライトが使用可能の場合は、「支援 (CH)」ボタンがプレイヤーの下に表示されます。
