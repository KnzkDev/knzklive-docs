# FAQ

## 外部インスタンスから投稿したコメント(トゥート)が表示できない

KnzkLive では、主となる Mastodon インスタンスから取得した **ハッシュタグタイムライン** と、 **独自のコメントサーバ** の二つからコメントとして表示しています。
そのため、ハッシュタグタイムラインから外部インスタンスのトゥートを取得する事ができますが、
**主となる Mastodon インスタンスのタイムラインから読み取れないアカウント**(誰からもフォローされていないか、サスペンドやサイレンスされている) は設計上コメントとして取得できませんのでご了承ください。

<Note>

KnzkDev が運営する [live.knzk.me](https://live.knzk.me) では、**Knzk.me** が主となる Mastodon インスタンスとして設定されています。

</Note>

## 一部環境で正常に視聴できない

現時点で、 **macOS 版 Safari** と **iOS 全般** において、KnzkLive における正常な視聴ができないことを確認しています。
これ以外で正常に視聴できない場合はお問い合わせください。
